Contacted

TypeScript PaLM2 · OAuth2 Expo · React Native · Tailwind May 2023 - Present

Contact Resource Manager

Related Links: Github Coming Soon to App Store

This CRM application is an all encompassing experience for reaching out and maintaining connections with professionals, family, and friendly contacts.

Using the OAuth2 API, all users may sign in with their Google Account. Users may import contacts from their phone or enter them manually, adding additional information to create a well rounded character which can be used to create more personal messages. Integrated with Google's PaLM2 API, Contacted utilizes its large language model to auto generate messages that the user may send to any contact in the application. The user may also dictate a time frequency, after which they will be reminded via push notifications that they should leave a friendly message to that person. A user may send messages straight from the application, or copy paste a message to a third party app.

Features

  • OAuth2 API Social authentication
  • PaLM2 API AI generated messages
  • Google Firestore Dynamically update app data
  • Expo Push Notifications User engagement
