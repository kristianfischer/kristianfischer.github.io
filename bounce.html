Bounce!

Java Google Ad Mob · Krita Android Studio · Game Dev November 2020 - March 2021

Mobile Game

Related Links: Github Google Play Store

This Android mobile application enlists the users touch to control a character through a series of gates, bouncing off moving walls on either side of the screen.

Bounce! is a full construction application including graphic design, audio incorporation, as well as classic game development. The games overarching theme of chalk and chalkboard was designed with Krita, with the associated audio elements being recorded and implemented into the app. Using Java's onDraw method with Canvas, the application is able to render multiple screens before and after starting the game. The inclusion of advertisements via Google Ad Mob generated minor revenue for the application, with over 10 users.
